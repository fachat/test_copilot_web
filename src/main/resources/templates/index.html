Quarkus Web Application Skeleton

Welcome to the Quarkus Web Application!

This is a skeleton web application demonstrating:

  • Quarkus Framework with JakartaEE
  • PostgreSQL Database Integration
  • RESTful API Services (JAX-RS)
  • Flyway Database Migrations
  • OAuth 2.0/OIDC Authentication
  • Server-Side Rendering with Qute
Get Started

Features

RESTful API

Access user data via REST API at /api/users

Database

PostgreSQL integration with JPA/Hibernate and Flyway

OAuth 2.0/OIDC

Secure authentication with OpenID Connect

Server-Side Rendering

Dynamic HTML pages with Qute templates
